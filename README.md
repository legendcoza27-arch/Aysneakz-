# Aysneakz E-Store

A full-featured e-commerce platform for sneakers with a React frontend and Python backend.

## Project Structure

```
Aysneakz-/
├── frontend/          # React application
├── backend/           # Python Flask API
└── README.md
```

## Features

### Frontend (React)
- Product catalog with listing
- Product detail pages
- Shopping cart functionality
- Search and filters
- User authentication
- Checkout process
- Order history

### Backend (Python)
- Admin authentication
- Product CRUD operations
- Image upload functionality
- RESTful API endpoints
- Database management
- Order processing

## Quick Start

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

## Environment Variables

See `.env.example` files in both frontend and backend directories.
